# Ducati Panigale V4 - Landing Page

Una landing page moderna e reattiva dedicata alla Ducati Panigale V4. Questo progetto è stato realizzato per mettere in pratica le competenze di base di web development, focalizzandosi sul responsive design e sulla cura dei dettagli estetici.

## 🚀 Anteprima Live
Puoi vedere il sito funzionante qui:
[LINK_DEL_TUO_SITO_GITHUB_PAGES]

## 🛠 Tecnologie Utilizzate
* **HTML5**: Struttura semantica del sito.
* **CSS3**: Stile, animazioni e layout.
* **Responsive Design**: Utilizzo di Flexbox e CSS Grid per adattare il sito a ogni schermo (mobile e desktop).

## 💡 Caratteristiche del progetto
* **Design Responsive**: Il layout si adatta fluidamente da mobile a desktop.
* **Componenti UI**:
    * **Hero Section**: Impatto visivo immediato.
    * **Griglia Modelli**: Layout a griglia (Grid) per mostrare i diversi modelli.
    * **Sezione Storia**: Layout split-screen per narrare l'heritage del brand.
* **Branding**: Utilizzo strategico del colore (accenti in rosso Ducati) per mantenere l'identità del marchio.

## 📁 Struttura del progetto
Il progetto è suddiviso in:
* `index.html`: Struttura principale della pagina.
* `style.css`: Foglio di stile dedicato per il responsive design e il branding.
